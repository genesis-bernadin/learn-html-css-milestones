Learning HTML

“If You Think Math is Hard Try Web Design”

In order to create websites, you must learn HTML which is the fundamental technology used to define the structure of a webpage. HTML is used to determine whether your web content should be recognized as a paragraph, list, heading, link, image, multimedia player, form, or one of many other available elements or even a new element that you define.

Once you have mastered HTML , you can move onto CSS and JavaScript.

  • CSS: shows you how to use it to style HTML. For example, CSS enables you to change:
    1. text size and fonts
    2. animations
    3. borders
    4. drop shadows
    5. layout
  • JavaScript: demonstrates how to implement interesting and dynamic functionality to web pages. For example:
    1. save users' history
    2. find users' location on a map
    3. add buttons
